# SkillAura Frontend

Welcome to the SkillAura Frontend application.

## Authentication Architecture
- **Email/Password Login:** Secure password-based authentication.
- **Google Sign-In:** OAuth integration for quick access.
- **JWT Authorization:** Secured endpoints using short-lived access tokens and refresh token rotation.
- **Auto-Login:** Users are automatically logged in upon successful registration.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Framer Motion

## Running Locally
```bash
npm install
npm run dev
```
